# DFIR KQL Threat Hunting Queries

Collection of KQL queries used for **Digital Forensics and Incident Response (DFIR)** and **threat hunting** investigations.

These queries are designed for use with:

• Microsoft Sentinel  
• Microsoft Defender XDR  
• Microsoft Defender for Endpoint  

---

## Detection Coverage

This repository includes KQL queries for detecting:

- Data exfiltration activity
- DNS exfiltration
- Password spraying attacks
- Suspicious PowerShell execution
- Phishing emails
- Suspicious email attachments
- Risky sign-ins
- Unusual outbound connections

---

## Example Use Cases

| Detection | Description |
|--------|--------|
| Data Exfiltration | Identify unusual outbound connections and possible data leakage |
| Password Spray | Detect repeated authentication failures across accounts |
| DNS Exfiltration | Identify high-frequency DNS queries |
| Suspicious PowerShell | Detect encoded or malicious PowerShell commands |
